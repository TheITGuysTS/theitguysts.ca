# The IT Guys – Technical Solutions Inc.

Public website for The IT Guys – Technical Solutions Inc.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This site is configured for GitHub Pages using GitHub Actions.

After files are pushed to the `main` branch:

1. Open the GitHub repository settings.
2. Go to **Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Confirm the custom domain is `theitguysts.ca`.
5. Ensure HTTPS is enforced after the certificate is issued.

DNS should be managed through Cloudflare.
